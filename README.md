# PinGenerator
I was having trouble making a pin, So i got the idea to make a random pin generator.
# Installation
1. Download The Files (Click The V Code Button)
2. Extract The Files
3. Add PinGenerator to your path
4. Open Cmd
5. Then type in the command {pingen}
